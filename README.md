# Windy City Dumpsters — Astro Site

## Local development
npm install
npm run dev

## Deploy
Connected to Netlify via GitHub. Push to main = auto-deploy.

## Adding a new city
1. Add entry to src/data/cities.json
2. Push to GitHub — Netlify auto-generates all pages

## Search Console verification
In src/layouts/BaseLayout.astro, replace REPLACE_WITH_YOUR_TOKEN with your verification token.
